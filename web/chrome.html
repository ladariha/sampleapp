NetBeans 7.3

Menu

 

Chrome plugin

  • Task: Running page without extension
    1. Have Chrome without IDE extension
    2. Run some file in Chrome
    • Result: NetBeans shows dialog saying you have to install the extension and gives you information on how to do it
  • Task: Running page with extension
    1. Have Chrome with IDE extension
    2. Run some file in Chrome
    • Result: Page is opened in Chrome, you can see extension's icon in Omnibar. Navigator window is populated
  • Task: Extension settings
    1. Have Chrome with IDE extension
    2. Run some file in Chrome
    3. Click on NetBeans icon in Omnibar
    • Result: Menu is opened, you can adjust window size. If you click on Customize you can add custom size and modify visibility settings. If you check the Selection Mode, selection mode is turned on
  • Task: Running page with disabled extension
    1. Have Chrome with IDE extension, go to Chrome's Settings -> Extensions and disable IDE extension
    2. Run some file in Chrome
    • Result: IDE shows warning message that you have to enable the extension (see issue 217004)
  • Task: Start Select Mode from contextual menu
    1. Have Chrome with IDE extension
    2. Run some file in Chrome
    3. Invoke right click contextual menu and select Start Select Mode
    • Result: Select Mode is enabled, you can select HTML elements from with browser. If you invoke contextual menu again, you can use Stop Select Mode to disable it
